이 코드에서 awesomeness를 클릭해서 연결하고 싶은 subpage를 정리한 ppt file이야. 
먼저 awesomeness를 클릭하면 slide 1과 같은 subpage로 연동되게 해줘. upload된 ppt file에서 점선으로 표시된 box는 웹사이트에 표시되는 것이 아니라, 너에게 설명을 위해 표시한 것이니 design에 포함시키지 마. 
Slide 1:
Check box라고 표시된 영역에 있는 네모 box에는 선택을 할 수 있도록 check box 형태로 구성하고, user가 항목을 선택할 수 있게 해. 그리고 input box에는 user가 직접 값을 입력 할 수 있도록 할거야. 하단에 lock과 unlock button이 있는데, 한번 입력한 값을 고정하고 싶을 때 항목 앞에 check box에 check를 하고 lock button을 누르면 다음에 사용할 때도 그 값이 저장되어 있는 기능이야. 예를 들어 unit weight를 계속 저장해 놓고 싶으면 unit weight의 input box에 값 입력 후 check box에 check한 후 lock을 누르면 값이 고정되는 거지. 이렇게 고정된 값을 수정하고 싶으면 check box에 check한 후 unlock을 누르고 input box에 값을 입력하면 되는거야. soil properties 입력하는 slide 1에 값들이 다 입력되면 save button을 누르면 입력한 값들이 google cloud pltform의 cloud run의 예측 모델의 입력값으로 전달이 될거야. gcp의 cloud run과 연동하는 것은 나중에 내가 다시 요청할꺼니까 지금 당장 code를 구성해줄 필요는 없어. soil properties가 다 입력이 되고 나면 next button을 누르면 slide 2와 같은 page로 넘어가게 되는거야.
Slide 2:
slide 2는 structure properties를 입력하는 page고 각각의 기능들은 slide 1과 동일해. slide 2의 next button을 누르면 slide 3과 같은 페이지로 넘어가게 되는거야.
Slide 3:
slide 3도 slide 1, slide 2와 동일하게 작동하게 되고, next button을 누르게 되면 slide 4로 넘어가게 되는거야.
Slide 4:
Slide 4에서는 예측하고 싶은 지점을 지정하는 page야. monitoring point의 x과표와 Y좌표를 input box에 입력할 수 있고, monitoring point를 추가 하고 싶으면 + add monitoring point 버튼을 누르면 monitoring 10까지 추가할 수 있어. monitoring point를 다 지정하고 지정된 지점을 고정하고 싶으면 lock 버튼을 누르면 되고 고정된 point의 x, y 값을 수정하고 싶으면 unlock 버튼을 누르고 수정할 수 있어. 포인트 들을 다 저장하고 싶으면 save 버튼을 누르면 되고, 선택된 지점을 삭제하고 싶으면 check box에 check한 후 delete button을 눌러서 삭제할 수 있어.
monitoring point가 다 지정되면 predict button을 누르고 다음 page로 이동할 수 있어.
Slide 5:
Slide 5에서는 slide 4에서 지정한 monitoring point들이 그대로 연결되서 보여지고, 각 지점의 예측 결과값들이 prediction value에 표시되는거야. 이 prediction value들은 slide 1, 2, 3에서 user가 입력한 값을 cloud run으로 전송되고, cloud run에 저장된 예측 모델에 입력값으로 input된 후 나온 예측 모델 결과값들을 받아서 보여줄 수 있도록 할거야. 이 기능은 당장은 필요없으니까 cloud run과 연동하는 code를 지금 당장 보여줄 필요는 없어. factor of safety의 결과도 마찬가지고 reliability도 마찬가지야. visualize button을 누르면 자동으로 slide 6으로 넘어가는거야.
Slide 6: 
Slide 6은 slide 1, 2, 3에서 입력한 값들을 예측 모델에 입력했을 때, 도출되는 contour image를 보여주는 페이지인데 이것도 cloud run과 연동 예정이고, 지금 당장은 보여줄 필요는 없어. total displacement의 contour image를 보여주는 페이지고 next button을 누르면 slide 7로 이동할 수 있어.
Slide 7은 slide 6과 동일하게 cloud run에서 실행한 Horizontal displacement contour image 결과를 보여줄 수 있는 page이고 이전 페이지로 돌아가고 싶으면 prev button, 다음 page로 이동하고 싶으면 next button을 누르면 되는거야.
Slide 8:
Slide 8은 slide 6, 7과 같이 cloud run에서 실행된 contour image 결과를 보여주는 page고, slide 8에 있는 prev button과 next button은 slide 7과 동일해.
Slide 9도 마찬가지이고 마지막에 Home button을 누르면 첫 home page로 돌아올 수 있어.

모든 slide에 있는 Ad Banner는 당장 필요한 것은 아니니까 자리만 비워둘 수 있게 배치해줘.
